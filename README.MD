## Install
```
npm i ydzy-ui
```

### 安装 pnpm

npm i pnpm -g

### 安装所有依赖

pnpm install

### 启动本地测试项目

pnpm run exm:dev

### 修改发布版本

 pnpm version patch

### 发布版本

pnpm publish --access public
